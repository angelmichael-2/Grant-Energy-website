# Grant Energy Website

This is a responsive business website built using HTML, CSS, and JavaScript for Grant Energy Resources, a diesel supply and energy services company.

## Features
- Responsive design (mobile and desktop friendly)
- Client request/contact form
- Clean and professional user interface
- Fast and accessible layout

## Technologies Used
- HTML5
- CSS3
- JavaScript
